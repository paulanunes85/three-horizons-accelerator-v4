# H3 Innovation Templates

AI/ML and advanced automation templates for innovative workloads.

## Available Templates

| Template | Description | Complexity |
|----------|-------------|------------|
| `ai-evaluation-pipeline` | Model evaluation framework | Complex |
| `copilot-extension` | GitHub Copilot extensions | Medium |
| `foundry-agent` | Azure AI Foundry agents | Complex |
| `mlops-pipeline` | Complete ML pipeline | Complex |
| `multi-agent-system` | Multi-agent orchestration | Complex |
| `rag-application` | RAG application scaffold | Complex |
| `sre-agent-integration` | SRE automation integration | Medium |

## Template Details

### ai-evaluation-pipeline

Creates model evaluation framework with:
- Evaluation metrics
- A/B testing setup
- Model comparison
- Performance tracking

### copilot-extension

Creates GitHub Copilot extension with:
- Extension scaffolding
- Custom actions
- Context integration
- Testing framework

### foundry-agent

Creates Azure AI Foundry agent with:
- Agent definition
- Tool integrations
- Prompt templates
- Deployment configuration

### mlops-pipeline

Creates complete ML pipeline with:
- Training pipeline
- Model registry integration
- Deployment automation
- Monitoring setup

### multi-agent-system

Creates multi-agent orchestration with:
- Agent definitions
- Coordination patterns
- Communication protocols
- State management

### rag-application

Creates RAG application with:
- Document ingestion
- Vector store setup
- Retrieval configuration
- Response generation

### sre-agent-integration

Creates SRE automation with:
- Alert integration
- Runbook automation
- Self-healing patterns
- Escalation logic

## Prerequisites

H3 templates typically require:
- Azure OpenAI service deployed
- AI Search service deployed
- Appropriate quotas for AI models

## Usage

Select any template from the RHDH portal under "Create" → "Choose a Template" → "H3 Innovation".

Ensure H3 infrastructure (AI Foundry) is deployed before using these templates.
