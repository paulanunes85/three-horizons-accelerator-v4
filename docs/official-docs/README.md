# Official Documentation Reference

This directory contains official documentation for all technologies used in the Three Horizons Accelerator. PDFs are stored locally (not committed to Git) for offline reference.

## Structure

```
official-docs/
├── ansible/            # Red Hat Ansible Automation Platform / AWX
├── argocd/             # ArgoCD GitOps
├── backstage/          # Backstage (upstream IDP)
├── kubernetes/         # Kubernetes / AKS
├── openshift/          # OpenShift / ARO
├── prometheus-grafana/ # Prometheus, Grafana, Alertmanager
├── rhdh/               # Red Hat Developer Hub
└── terraform/          # Terraform / Azure Provider
```
